XBinaryViewer

## Building

This project is configured to **always build in a temporary system folder** to keep the source tree clean.

For detailed build instructions, see [BUILD.md](BUILD.md).

Quick start:
```bash
cd build
./build_linux.sh        # Linux
./build_macos.sh        # macOS
build_windows_msvc2019_x64.bat  # Windows
```

Built packages will be available in the `packages/` directory.
